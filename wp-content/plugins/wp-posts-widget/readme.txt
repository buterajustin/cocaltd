Posts widget! Add list of recent posts from your CUSTOM POST TYPE.

== Description ==

It's a very simple plugin for display to recent posts list in sidebar widget.You can choose custom post type for display to list posts in sidebar.

Note : please don't forget to leave your valuable feedback and suggsations for make my plugin much more better compare to other plugins.

 Send your query on raghunath.0087@gmail.com


= `Features` = 
 * An option for show comment count
 * An option for define hide widget title
 * An option for choose to your custom post type
 * An option for exclude specific posts.
 * An option for define post filter order.
 * An option for define to number of posts
 
 
== Frequently Asked Questions ==

1.) Can i hide widget title?
csv
 * Yes, You can hide widget title if you do not show in sidebar.
 
2.) Can i show list of posts from my custom post type?

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The categories post will be available on widget "WP Posts Widget" (Apperance >> Widget)
